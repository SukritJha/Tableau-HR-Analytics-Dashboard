HR Analytics Dashboard (Tableau)
This repository contains the Tableau workbook and associated data for an interactive HR Analytics Dashboard. The project analyzes a comprehensive, AI-generated dataset of employee information to provide key insights into workforce demographics, compensation, and performance.

The dashboard is split into two main views: a high-level HR Summary and a granular Employee Details table.

üìã Project Overview
The primary goal of this project is to build a user-friendly tool for HR managers and leadership to:

Understand the company's demographic makeup (gender, education, location).

Analyze key HR metrics like total employees, average salary, and performance.

Quickly filter and explore employee data by department, location, and other attributes.

Provide a detailed, searchable list for finding specific employee information.

‚ú® Key Features & Visualizations
The workbook includes two main dashboards:

1. HR Summary Dashboard
A high-level overview of the entire workforce, featuring:

KPI Cards:

Total Employees

Total Departments

Average Salary

Average Performance Rating

Visualizations:

Employee Distribution by State: A geographic map showing the concentration of employees across the US.

Employee Count by Department: A bar chart comparing the size of different departments.

Employee Count by Gender: A pie chart showing the gender distribution.

Employee Count by Education Level: A bar chart breaking down the workforce by educational attainment.

Employee Count by Performance Rating: A bar chart showing the distribution of performance scores.

Interactive Filters:

State

Department

Education Level

Gender

Performance Rating

2. Employee Details
A detailed table view for drilling down into specific employee data.

Detailed Table:

Shows Employee_ID, Full Name, Department, Job Title, Hiredate, and Salary.

Interactive Filters:

Department

Job Title

Year (of Hiredate)
